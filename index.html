About the Project

Design and development of a Computer Numerical Control machine capable of performing accurate two-dimensional cutting, plotting, and engraving operations. The system integrates mechanical, electrical, and software modules to automate machining tasks with improved productivity and precision.

Microcontroller control and G-Code automation handle trajectory execution, giving the machine a framework directly expandable to PCB etching and laser applications.

// 02

Project Objectives

01 Design a CNC based cutter for 2D motion control.
02 Enable automated trajectory following using G-Code.
03 Demonstrate cutting and engraving capabilities.
04 Establish a framework expandable to PCB etching and laser applications.
// 03

System Components

Mechanical Structure

Aluminium profiles support the X, Y and Z axes. Slide rails and lead screws enable motion.

Electronics

Arduino Uno, CNC Shield V3, and A4988 stepper drivers controlling NEMA 17 stepper motors.

Software Tools

Aspire 10 for CAD/CAM, GRBL controller for G-Code execution, and Arduino IDE for firmware.

Actuation & Motion

Stepper motors provide precise motion control with micro-stepping up to 1/16th resolution.
